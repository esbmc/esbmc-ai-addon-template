# ESBMC-AI Addon Template

Template repository for creating custom [ESBMC-AI](https://esbmc.github.io/esbmc-ai/) addons. ESBMC-AI is an AI-powered framework for program verification and analysis that leverages LLMs for code analysis, verification, and repair.

## Usage

1. Click "Use this template" to create your addon repository
2. Search for `TEMPLATE:` comments in the code and customize them
3. Rename the `my_addon/` directory to your addon name
4. Update imports and references throughout the codebase
5. Implement your addon logic in `commands/` and `verifiers/`

See template files for detailed examples and inline documentation.
